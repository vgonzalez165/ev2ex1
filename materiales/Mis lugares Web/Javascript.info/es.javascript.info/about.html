    Acerca del proyecto

    El tutorial de JavaScript moderno fue creado en 2007 por Ilya Kantor, y es actualizado regularmente desde entonces. Nuevos capítulos son agregados, los obsoletos quitados, y así se mantiene fresco. La versión en PDF es de unas 1300 páginas, comenzando desde el principio para luego continuar con tópicos más avanzados.

    Es un libro, no un video, siendo que para muchas personas la lectura es más rápida. Además, los libros son más fáciles de actualizar y mantener modernos :).

    El contenido de este tutorial es open source y todos son bienvenidos a contribuir.
